# Contributing

This repo is mainly the **special profile repository** (`username/username`). Changes should stay focused on the profile README, workflows, and assets.

## Recommended workflow (branch → PR → merge)

1. Create a branch from `main`.
2. Commit your edits.
3. Open a **pull request** into `main` (even as the repo owner — use *Compare & pull request* after pushing).
4. Merge the PR when ready (**Squash and merge** or **Merge** — both count as a merged PR for GitHub profile achievements).

Direct pushes to `main` work but do **not** add to achievements that require merged pull requests.

## Pair programming (optional)

To record shared work in git history, use [co-authored commits](https://docs.github.com/pull-requests/committing-changes-to-your-project/creating-and-editing-commits/creating-a-commit-with-multiple-authors) on commits that land via a merged PR.

## Discussions (Galaxy Brain)

GitHub achievements for accepted answers use **Discussions**. Enable them once:

**Repository → Settings → General → Features → Discussions** → turn on, then add a **Q&A** category.

After that, substantive answers can be marked as the accepted solution.
